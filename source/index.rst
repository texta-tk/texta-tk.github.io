Texta Toolkit
=============

TEXTA Toolkit is an open-source software for building RESTful text analytics applications.
It is being developed by `TEXTA Ltd <https://www.texta.ee>`_.
This documentation contains information about installing and running **TEXTA Toolkit 2.x**.

The source code for Toolkit's Django REST API is available at `git.texta.ee/texta/texta-rest <https://git.texta.ee/texta/texta-rest>`_.

The source code for Toolkit's Angular GUI is available at `git.texta.ee/texta/texta-rest-front <https://git.texta.ee/texta/texta-rest-front>`_.
